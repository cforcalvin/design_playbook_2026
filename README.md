# UI Pattern Playbook

A single-file, AI‑ready UI pattern playbook for generating premium, consistent UI patterns for a voice‑agent builder.

## What’s included
- `ui-pattern-playbook.md` — standalone playbook with deterministic rules, checklists, and examples.
- `example.html` — static components aligned to the playbook rules.
- `example2.html` — modal + table example (selectable rows).
- `example3.html` - chat window example.

## How to use
- Provide `ui-pattern-playbook.md` as system context to an AI model.
- Prompt the model to output patterns using the required template in the playbook.

## Compatibility
Designed to work with Cursor, Codex, and Claude using plain Markdown only.
